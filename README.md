E&M-Joyeria 💎
Sitio web oficial de E&M-Joyeria, un emprendimiento dedicado al diseño y creación de prendas exclusivas hechas con materiales de alta calidad. El proyecto es una plataforma informativa y de contacto diseñada para ofrecer una experiencia elegante y moderna a los usuarios.

🚀 Características
Diseño Responsive: Optimizado para dispositivos móviles, tablets y computadoras de escritorio mediante Bootstrap 5.

Animaciones Fluidas: Uso de la librería AOS (Animate On Scroll) para efectos visuales al desplazar la página.

Interactividad: * Carrusel de imágenes dinámico para productos destacados.

Formulario de contacto funcional.

Integración con Google Maps.

Botón flotante de WhatsApp para atención directa.

Multimedia: Sección de video embebido de YouTube explicando el proceso de fabricación.

🛠️ Tecnologías utilizadas
HTML5: Estructura semántica del sitio.

CSS3: Estilos personalizados (almacenados en css/style.css).

Bootstrap 5: Framework para el diseño y componentes responsivos.

Google Fonts: Tipografías personalizadas (El Messiri, Kanit, Merienda, etc.).

AOS Library: Animaciones de entrada de elementos.

FontAwesome / Iconos: Para redes sociales y botones de contacto.

📂 Estructura del Proyecto
El sitio se divide en las siguientes secciones:

Inicio (index.html): Vista principal con carrusel, tarjetas de categorías (Anillos, Collares, etc.) y sección de video.

Galería (galeria.html): Muestra fotográfica detallada de los productos disponibles.

Nosotros (nosotros.html): Historia del emprendimiento y visión de los fundadores.

Servicios (servicios.html): Información sobre métodos de envío y procesos de compra.

Contacto (contacto.html): Formulario para consultas y mapa de ubicación.

Login (inicio.html): Interfaz de inicio de sesión para clientes.

📦 Instalación y Uso
Para visualizar el proyecto localmente:

Clona este repositorio o descarga el archivo ZIP: git clone https://github.com/tu-usuario/nombre-del-repo.git

Asegúrate de mantener la estructura de carpetas:

/imagenes: Contiene todos los assets visuales.

/css: Contiene la hoja de estilos.

/js: Contiene los archivos JavaScript.

Abre el archivo index.html en cualquier navegador moderno.
